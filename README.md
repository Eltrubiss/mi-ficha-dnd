# 🐉 Ficha de Personaje - D&D 5e Web App

¡Bienvenido a la aplicación web de gestión de personajes para Dungeons & Dragons 5ta Edición! 

Esta es una herramienta digital, ágil y ligera, diseñada para que los jugadores puedan crear, editar y llevar el seguimiento de sus personajes de rol de forma sencilla, rápida y directamente desde el navegador.

---

## 🚀 Cómo usar la aplicación

1. **Accede a la app:** Ingresa a la aplicación a través de este enlace: [https://eltrubiss.github.io/mi-ficha-dnd/](https://eltrubiss.github.io/mi-ficha-dnd/)
2. **Crea tu héroe:** Haz clic en **Añadir Personaje** en la pantalla principal.
3. **Gestiona tus estadísticas:** Rellena tu clase, nivel, puntuaciones de características y vida. La interfaz realizará los cálculos de modificadores de forma automática.
4. **Exporta y respalda:** Utiliza el botón de **Exportar** (ícono de descarga) para descargar tu ficha en formato `.json` a tu computadora o celular.

## 💾 Sobre el guardado de datos (¡Importante!)

Para garantizar que la aplicación sea rápida y privada (sin necesidad de contraseñas ni bases de datos externas), **todo tu progreso se guarda localmente en el almacenamiento de tu navegador (localStorage)**.

> **⚠️ ADVERTENCIA:** Si limpias la caché de tu navegador, borras el historial de navegación, o entras desde una ventana en modo "Incógnito/Privado", **tus personajes se borrarán**. Por favor, acostúmbrate a utilizar el botón de "Exportar" al final de cada partida para guardar a tu personaje en un archivo seguro en tu dispositivo.

## 📱 Diseño Adaptable (Mobile First)

¿Juegas en una mesa física y solo tienes tu teléfono a mano? ¡No hay problema! La interfaz de esta aplicación ha sido diseñada desde cero para ser *Mobile First*, lo que significa que los menús, las cuadrículas y las barras de vida se adaptarán perfectamente a la pantalla de tu celular sin perder legibilidad.
